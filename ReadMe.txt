In order to run the code you must have python installed. 
Go into your command terminal and change the directory to where you put the "Question" files in. 
After you are in the correct directory, type the file name of the question you wish to look at.

Example: 

C:\Users\Johny Xiong\Desktop>cd PythonProj

C:\Users\Johny Xiong\Desktop\PythonProj>Question1.py
